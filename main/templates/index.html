Append dimension you want as placeholder like this: http://i.khophi.co/100x100. The myaddress can be a local instance of this app on your computer or via this website address.
Vary colors of the placeholder images by specifying the RGB values like this: http://i.khophi.co/100x100/c/255.0.0. REMEMBER to include the '/c/' part of the URL. The 'c' stands for Color
